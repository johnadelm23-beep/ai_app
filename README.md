# ✨ Overview

Janjony AI is a modern Flutter AI chat application that provides a smooth conversational experience with a clean and responsive UI.

The application supports:
- AI-powered conversations
- Multiple chat sessions
- Professional drawer navigation
- Authentication screens
- Smooth user experience
- Modern UI inspired by Gemini

# 🚀 Getting Started

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/janjony-ai.git
```

---

## 2️⃣ Install Dependencies

```bash
flutter pub get
```

---

## 3️⃣ Run the App

```bash
flutter run
```

---

# 🔑 Gemini API Setup

Create your API key from Google AI Studio:

👉 https://aistudio.google.com/app/apikey

Then add it inside:

```dart
GeminiService.dart
```

```dart
static const String apiKey = "YOUR_API_KEY";
```

---

# 📦 Build Release APK

```bash
flutter build apk --release
```

APK location:

```bash
build/app/outputs/flutter-apk/app-release.apk
```
